Este é um protótipo que utiliza singleton e troca de mesagens para desacoplar o código da camada de interface grafica do código das funcionalidades e mecanicas do jogo. Neste exemplo é utilizado uma simulação de um inventario que incrementa e decrementa informações sobre ouro coletado e ouro possuido. A troca de mensagens entre as diferentes camadas do software faz com que o código do inventario não esteja fortemente acoplado ao código da GUI.


This is a prototype that uses singleton and exchange of mesagements to decouple the interface code layer from the code of the functionalities and mechanics of the game. In this example a simulation of an inventory is used that increases and decreases information about gold collected and gold possessed. The exchange of messages between the different layers of the software means that the inventory code is not strongly coupled with the GUI code.
