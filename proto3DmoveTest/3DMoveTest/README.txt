Este é um protótipo de movimentação 3D. Todos os códigos foram implementados por mim e o modelo 3D da personagem
utilizada pode ser encontrada na Asset Store do Unity buscando por UnityChan.

This is a 3D drive prototype. All codes have been implemented by me and the 3D model of the character
used can be found in the Unity Asset Store by UnityChan.


Commands:

keyboard commands

walk - w,a,s,d e arrows
e - lantern
q - inventary
press left mouse e roll to move cam
esc - quit

-------------------------------------------
controller commands

walk - left axis
RB - lantern
select - inventary
start - quit
move cam - right axis
